Privacy Policy

Privacy Policy for Goal Guru Effective Date: 20th October 2023

1. Information We Collect Goal Guru collects the following information from its users: Full name Email address Anonymized device information This information is collected at the time of sign-up and is stored securely within an encrypted database. All passwords are hashed and encrypted for added security.

2. Purpose of Data Collection We collect this information to provide you with access to the Goal Guru app. Without this information, you would not be able to use our app.

3. Use and Storage of Collected Data We use the collected data solely for the following purposes: Account verification Facilitating the creation of games and groups with other players All user data is stored securely within an encrypted database.

4. Data Sharing Goal Guru does not share user information with any third parties.

5. User Control Over Data Users have the ability to delete their Goal Guru account in two ways: By sending an email to Joew795@gmail.com with the subject "Goal Guru account deletion." By using iCloud to revoke access.

6. Data Security To protect user data, we employ the following security measures: Passwords are hashed and encrypted using the latest encryption technologies. Databases are encrypted using state-of-the-art encryption technology. Any further user information stored on the device is anonymized. All user information becomes obsolete upon deleting the app.

7. Contact Information If you have any questions or concerns related to your privacy or this privacy policy, please contact us at joew795@gmail.com.
